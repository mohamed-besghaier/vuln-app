# vuln-app

Simple Node.js and Express app backed by SQLite.

This project is a work in progress.

## Stack

- Node.js
- Express
- SQLite

## Project structure

```text
vuln-app/
├─ app.js
├─ package.json
├─ db/
├─ models/
├─ pages/
├─ public/
├─ routes/
└─ uploads/
```

## Run locally

```bash
npm install
node app.js
```

The app runs on `http://localhost:3000`.
